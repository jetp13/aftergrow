🌘 Afterglow

回到身體，回到自己。

有些夜晚，人不是迷失，而是 太滿。
身體、情緒、連結、慾望同時被推到一個邊緣，來不及回過神。

Afterglow 在這樣的情境裡誕生：
不是要你改變什麼、戒掉什麼、成為更好的人。
它只是先讓你 回得來。
只要有呼吸，你就仍然有選擇。

這是一個 呼吸引導 + 緊急支援 + 權利意識 的 Web App。
溫柔，但不逃避現實；冷靜，但不失去人味。

⸻

🪐 核心模組

模組	它在做的事	狀態
Breathing 呼吸引導	用呼吸球與語音，讓神經系統降速，回到可感的身體	✅ 已上線
Emergency 緊急支援	在事情「還來得及」時給你判斷與止損的路徑	🛠 開發中
Know Your Rights 你有的權利	面臨臨檢 / 搜索時，提醒界線與清醒仍然存在	🛠 編輯中
Aftercare 派對後復原	引導身體回到人：睡眠、補水、神經修復	📝 設計中


⸻

🎧 使用方式
	1.	找一個你可以稍微停下來的地方
	2.	點開 呼吸引導
	3.	不用做得好、也不需要專注
	4.	只要一吸、一吐，把注意力放回自己

沒有要你「放空」。
你只是在 回家。

🌙 為什麼會有它

在那樣的房間裡、音樂、呼吸、光線、汗水之間——
很多人不是在追快樂，而是在暫時遠離痛。

我們大多不是第一次迷路，
但往往 沒有一條能回來的路。

Afterglow 想做的，是把那條路 先放在你手裡。
不是要你變好，只是要你 不必再一個人掉下去。

你值得被帶回來。
你值得在自己的身體裡，再次醒著。
